# 集合

- **无序的**、**不重复的**元素构成
- 特殊的数组
  - 无序、不能重复
  - 不能通过下标访问，元素具有唯一性

> ES6 自带 `Set` 类，这里使用 `object` 封装

## Set 封装

```javascript
class Set {
    constructor() {
        this.items = {};
    };
}
```

- `add(value)`：集合中添加一个新项
- `remove(value)`：集合中移除某个值
- `has(value)`：判断值是否在集合中
- `clear()`：移除集合中的所有项
- `size()`：返回集合中包含元素的数量
- `values()`：返回一个包含集合中所有值的数组
